## Development

```bash
git clone https://github.com/willnguyen1312/build-your-own-signals
cd build-your-own-signals
pnpm install
```

## Commands

### Run in browser

```bash
pnpm dev
```

### Run tests with Vitest in watch mode

```bash
pnpm test
```

### Run tests with Vitest

```bash
pnpm test:unit
```

### Run tests with Playwright

```bash
pnpm test:e2e
```
